# dotfiles

## Prerequisites

requires git, zsh, autojump, [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh)

```
brew install git zsh autojump
```

oh-my-zsh should be installed manually

```
git clone https://github.com/ohmyzsh/ohmyzsh.git ~/.oh-my-zsh

chsh -s $(which zsh)
```

## Quickstart

```
source sync.sh
```
